# Springboot + Shiro 实现认证和授权

> 项目来自[Springboot+Shiro 实现认证和授权](https://www.bilibili.com/video/av37984279/?p=12)


### 功能

- 认证(登录)功能
- 授权功能
- Thymeleaf整合权限标签，登录后只显示拥有有权限的功能
